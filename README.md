# Stroophy Digital Services (SDS) Website

Professional website for Stroophy Digital Services featuring dark/light themes, responsive design, and comprehensive digital infrastructure solutions.

## Features

- **Dark/Light Theme Toggle**: Users can switch between themes
- **Responsive Design**: Mobile-first approach works on all devices
- **Complete Page Structure**:
  - Homepage with service overview
  - Services page with detailed offerings
  - About page with company information
  - Portfolio page with project showcase
  - Contact page with working form
- **Tech Backgrounds**: Circuit patterns and gradient backgrounds
- **Professional Design**: Clean, modern, not startup-y
- **Truthful Content**: No fake stats or inflated claims

## Technology Stack

- **Next.js 15** with TypeScript
- **Tailwind CSS** for styling
- **React Icons** & **Lucide React** for icons
- **Dark Mode** with CSS variables
- **Responsive Design** with mobile-first approach

## Pages Structure

1. **Homepage** (`/`)
   - Hero section with gradient background
   - Service preview cards
   - Professional approach section
   - Call-to-action sections

2. **Services** (`/services`)
   - Detailed service descriptions
   - Construction Tech focus
   - Web Development services
   - Cloud Solutions
   - Digital Infrastructure

3. **About** (`/about`)
   - Company story (truthful, no fake history)
   - Team information
   - Company values
   - Melbourne-based focus

4. **Portfolio** (`/portfolio`)
   - Project showcase
   - Construction Management App (in development)
   - SDS Website (live)
   - Cloud migration projects
   - Digital infrastructure platforms

5. **Contact** (`/contact`)
   - Working contact form
   - Contact information
   - FAQ section
   - Response time information

## Design Principles

1. **Truthfulness**: No fake "100+ projects" or inflated experience claims
2. **Professionalism**: Established business presentation, not startup vibe
3. **Functionality**: All buttons and links lead to actual pages
4. **Accessibility**: Proper contrast, semantic HTML, keyboard navigation
5. **Performance**: Optimized images, efficient code, fast loading

## Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Start production server
npm start
```

## Deployment

The site is configured for Vercel deployment. Environment variables can be configured in Vercel dashboard.
